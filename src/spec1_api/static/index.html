Terminal shortcuts for working with SPEC-1 directly
Read latest brief
cat ~/spec-1/briefs/spec1_brief_latest.md
List all briefs
ls ~/spec-1/briefs/spec1_brief_*.md
Read specific date
cat ~/spec-1/briefs/spec1_brief_2026-04-24.md
Start API server
cd ~/spec-1 && export PYTHONPATH="src" && python -m spec1_engine.main
Run intelligence cycle manually
cd ~/spec-1 && export PYTHONPATH="src" && \
ANTHROPIC_API_KEY=$(cat .env | grep ANTHROPIC_API_KEY | \
cut -d= -f2 | tr -d '\r') python -m spec1_engine.app.cycle
